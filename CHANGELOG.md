# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project currently follows a lightweight, date-based release history.

## [0.1.0] - 2026-06-21

### Added
- Root Node.js API for cleaned Douyin live room data.
- Guest-cookie bootstrap, `a_bogus` signing, HTML fallback, and best-effort mobile time enrichment.
- Localhost-by-default auth behavior with optional `API_KEY` enforcement.
- Static and per-request upstream proxy support.
- Vercel deployment target under `vercel/`.
- Cloudflare Workers deployment target under `cloudflare/`.
- Basic CI syntax checks via GitHub Actions.
